# Full Stack File Upload

A simple full stack file upload project built using React, Express, and Multer.

This project allows users to:

- Select an image from frontend
- Upload image to backend
- Store uploaded image on server
- Display uploaded image in browser

---

# Tech Stack

Frontend:
- React
- Axios
- Vite

Backend:
- Node.js
- Express.js
- Multer
- CORS
